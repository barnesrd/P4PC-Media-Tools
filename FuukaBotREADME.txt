Put FuukaBot.exe in the StreamControl directory and launch.
This bot will take commands from twitchchat and change text on
stream labels from StreamControl.

Usage doc: https://pastebin.com/8UTkTzF5

This bot is only usable by mods or Jolteo_ (for debug purposes) at the moment,
more functionality will come with time.

CHALLONGE INTEGRATION:
- To configure challonge functions edit values in the .env file found in the StreamControl folder.

.env contents:
CHANNEL=[twitch channel name]

TOUNAMENT_ID=[challonge tournament extension]

CHALLONGE_USERNAME=[Username of user with view permission of tournament]
CHALLONGE_API_KEY=[API key of the user listed above]
